# Elderly Health Monitoring & Emergency Alert System

## Project Overview

The **Elderly Health Monitoring & Emergency Alert System** is an integrated solution designed to continuously monitor the vital health parameters of senior citizens living alone and provide instant alerts during emergencies. The system assists elderly individuals, their families, and caregivers by ensuring **real-time monitoring, early detection of health anomalies, and timely intervention**.

This project demonstrates the practical application of embedded systems, IoT sensors, and software engineering principles to build a **reliable and life-saving health monitoring solution**.

## Objectives

The primary objectives of this project are:

* To continuously monitor critical health parameters such as blood pressure, heart rate, oxygen levels, and blood sugar.
* To detect sudden falls and health anomalies promptly.
* To generate real-time alerts for caregivers and family members.
* To maintain historical health records for analysis and preventive care.
* To design a system that is user-friendly, scalable, and reliable.
  
## Key Features

* **Real-Time Health Monitoring**: Monitors blood pressure, heart rate, oxygen saturation, and blood sugar levels.
* **Fall Detection**: Detects sudden falls or abnormal movements and triggers instant alerts.
* **Emergency Notifications**: Sends alerts via mobile or web application to caregivers, family members, or medical professionals.
* **Historical Data Management**: Maintains past health records for trend analysis and preventive care.
* **User-Friendly Interface**: Simple interface designed for elderly users and their caregivers.
* **Remote Monitoring**: Allows family members and caregivers to monitor health status remotely.

## System Architecture

The system follows a modular architecture for maintainability and scalability:

* **Sensor Layer**: Includes blood pressure monitors, pulse oximeters, glucose sensors, and accelerometers.
* **Processing Layer**: Microcontrollers like Arduino or Raspberry Pi process sensor data and detect anomalies.
* **Communication Layer**: Sends alerts and notifications via Wi-Fi or GSM modules.
* **Application Layer**: Mobile/web dashboard for real-time monitoring, historical data visualization, and alerts management.


## Technologies Used

* **Hardware**: Arduino / Raspberry Pi, Blood Pressure Sensor, Pulse Oximeter, Glucose Sensor, Accelerometer
* **Software**: Python / Node.js (backend processing), React / HTML-CSS-JS (frontend dashboard)
* **Database**: MySQL / MongoDB (for storing historical health data)
* **Development Tools**: VS Code, Git, npm
* **Deployment Environment**: Localhost / Cloud-ready setup


## Installation and Setup

1. Clone or extract the project repository.
2. Connect the hardware sensors to the microcontroller.
3. Install required dependencies:

   ```
   npm install
   ```
4. Configure environment variables (if applicable).
5. Start the backend and frontend applications:

   ```

   ```
6. Access the monitoring dashboard via the configured local server or web interface.
